## 简介

叫到帮小程序就是帮助大家解决 🧰 各种问题的，其中包括室内装修、电器维修、货物搬迁等问题，有这些问题就找叫到帮！（此项目暂未完成 ❎ ）

## 安装使用

```bash
# 进入项目
cd HELPER_FACTION_official_weapp_customer

## 安装依赖
npm i

# development
npm run watch

# production
npm run build
```

使用小程序开发者工具打开项目文件夹 📁 下dist中对应平台的文件夹 📁 即可预览效果。
